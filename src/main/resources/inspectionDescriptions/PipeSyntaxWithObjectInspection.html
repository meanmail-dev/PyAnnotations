This inspection reports when the pipe union with 'object' can be simplified (Python 3.10+).
int | object → object
str | int | object → object

Since 'object' is the base class of all classes in Python, a union that includes 'object' is equivalent to just 'object'. This simplification makes the code more readable.

Note: This inspection only applies to Python 3.10+ pipe union syntax (X | Y). For classic Union[X, Y] syntax, see the UnionWithObjectInspection.
