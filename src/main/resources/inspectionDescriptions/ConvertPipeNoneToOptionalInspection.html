This inspection suggests converting X | None syntax to Optional for backward compatibility.
int | None → Optional[int]
str | None → Optional[str]
int | str | None → Optional[Union[int, str]]

The pipe syntax (X | None) for optional types was introduced in Python 3.10 (PEP 604). If your project needs to support Python 3.9 or earlier, you should use Optional[X] from the typing module instead.

Note: This inspection is disabled by default. Enable it when your project targets Python versions earlier than 3.10.
