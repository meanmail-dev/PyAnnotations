This inspection suggests converting classic Union syntax to modern pipe syntax (Python 3.10+).
Union[int, str] → int | str
Union[int, str, float] → int | str | float

Python 3.10 introduced PEP 604 which allows using the | operator for type unions. This syntax is more concise and doesn't require importing from typing.

Note: This inspection is disabled by default. Enable it when your project targets Python 3.10 or later and you want to modernize type annotations.
