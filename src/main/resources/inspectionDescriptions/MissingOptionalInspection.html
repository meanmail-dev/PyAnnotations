This inspection detects parameters with None as default value but without Optional in the type annotation.
def foo(x: int = None) → def foo(x: Optional[int] = None)
def bar(name: str = None) → def bar(name: Optional[str] = None)

When a parameter has None as its default value, the type annotation should include Optional (or X | None in Python 3.10+) to correctly indicate that the parameter can be None.

Note: This inspection is disabled by default. Enable it if you want strict type annotation checking in your project.
