This inspection reports redundant parentheses around single types in pipe unions (Python 3.10+).
(int) | str → int | str
bool | (float) → bool | float

Parentheses around a single type in a pipe union are unnecessary and can be removed to improve readability.

Note: Parentheses around nested unions like (int | str) | float are handled by a separate inspection (NestedPipeUnionInspection).
