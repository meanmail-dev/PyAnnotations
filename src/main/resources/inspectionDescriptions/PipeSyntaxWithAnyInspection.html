This inspection reports when the pipe union with 'Any' can be simplified (Python 3.10+).
int | Any → Any
str | int | Any → Any

Since 'Any' is compatible with all types in Python, a union that includes 'Any' is equivalent to just 'Any'. This simplification makes the code more readable.

Note: This inspection only applies to Python 3.10+ pipe union syntax (X | Y). For classic Union[X, Y] syntax, see the AnyInUnionInspection.
