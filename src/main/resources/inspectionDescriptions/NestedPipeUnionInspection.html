This inspection reports nested pipe unions that can be flattened (Python 3.10+).
(int | str) | float → int | str | float
bool | (int | str) → bool | int | str

Nested pipe unions with parentheses can be flattened into a single union expression. This makes the type annotation more readable without changing its meaning.

Note: The pipe operator | for unions is left-associative, so A | B | C is parsed as (A | B) | C. Explicit parentheses around inner unions are redundant.
