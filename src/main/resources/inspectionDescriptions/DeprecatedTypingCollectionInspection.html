This inspection suggests replacing deprecated typing module collections with built-in generic types (Python 3.9+, PEP 585).
List[int] → list[int]
Dict[str, int] → dict[str, int]
Set[str] → set[str]
Tuple[int, str] → tuple[int, str]
FrozenSet[int] → frozenset[int]
Type[MyClass] → type[MyClass]

Since Python 3.9 (PEP 585), built-in collection types like list, dict, set, and tuple can be used directly as generic types in annotations. The typing module equivalents are deprecated and may be removed in future Python versions.

Note: This inspection is disabled by default. Enable it when your project targets Python 3.9 or later.
