This inspection suggests converting pipe union syntax to classic Union for backward compatibility.
int | str → Union[int, str]
int | str | float → Union[int, str, float]

The pipe syntax (X | Y) for type unions was introduced in Python 3.10 (PEP 604). If your project needs to support Python 3.9 or earlier, you should use Union[X, Y] from the typing module instead.

Note: This inspection is disabled by default. Enable it when your project targets Python versions earlier than 3.10.
