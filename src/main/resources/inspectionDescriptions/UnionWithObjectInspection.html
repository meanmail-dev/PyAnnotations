This inspection reports when the Union[object, ...] annotation was used instead 'object'.
Union[object] -> object
Union[object, int] -> object
Optional[Union[object, string]] -> Optional[object]

In Python's type system, 'object' is the base class for all types. When 'object' appears in a Union:

  • The Union becomes equivalent to just 'object' since every type is already a subtype of 'object'
  • The other types in the Union become redundant
  • The annotation can be simplified for better readability and clarity

This inspection helps eliminate unnecessary complexity in type annotations.
