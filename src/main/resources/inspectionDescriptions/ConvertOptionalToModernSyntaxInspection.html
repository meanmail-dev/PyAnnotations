This inspection suggests converting Optional syntax to modern pipe syntax (Python 3.10+).
Optional[int] → int | None
Optional[str] → str | None

Python 3.10 introduced PEP 604 which allows using X | None instead of Optional[X]. This syntax is more concise and doesn't require importing Optional from typing.

Note: This inspection is disabled by default. Enable it when your project targets Python 3.10 or later and you want to modernize type annotations.
