This inspection reports when the Union[item] annotation was used instead 'item'.
Union[item] -> item

A Union type with only a single item is redundant and can be simplified:

  • Union[str] is equivalent to just str
  • Union[Dict[str, int]] is equivalent to Dict[str, int]
  • Union[List[int]] is equivalent to List[int]

This inspection helps maintain clean and concise type annotations by removing unnecessary Union wrappers around single types. The simplified form is:

  • More readable
  • More maintainable
  • Consistent with Python typing best practices
