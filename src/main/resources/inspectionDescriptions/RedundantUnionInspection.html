This inspection detects redundant types in Union where one type is a subtype of another.
Union[int, bool] → int (bool is subtype of int)
Union[float, int] → float (int is subtype of float)
Union[complex, float, int] → complex

In Python's type system, bool is a subtype of int, and int is considered a subtype of float for type checking purposes. When a Union contains both a type and its subtype, the subtype is redundant.
